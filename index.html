Recent Projects

IndexEase Application Software

Python

The Automated PDF Organization and Classification System is a Python-based desktop application designed to streamline the organization and retrieval of PDF documents. Using machine learning, OCR, and topic modeling, the system automatically classifies documents, extracts key information, and reorganizes content for improved accessibility. The intuitive GUI allows users to quickly access documents via an index, saving time and improving productivity. Built with PyMuPDF, Tesseract OCR, Gensim, and Scikit-learn, it is cross-platform compatible with both Windows and Linux, providing an efficient solution for managing large document collections.

Supplier Selection Optimization Algorithm

C++

The Supplier Selection Optimization Algorithm is a cutting-edge solution designed to enhance logistics and supply chain efficiency. Using a Genetic Algorithm (GA), this project optimizes the allocation of orders to suppliers and vehicles, reducing processing times and minimizing transportation costs. By considering factors such as supplier availability, vehicle capacity, and travel distance, the algorithm dynamically adapts to changing logistics requirements, ensuring timely deliveries and efficient resource utilization. Scalable and flexible, this system provides a robust tool for industries aiming to streamline their supply chain operations and improve decision-making processes. View on GitHub
